# Christopher De Jesus - Portfolio

Welcome to my personal portfolio website showcasing my work at the intersection of technology, psychology, and creative problem-solving.

**Live site:** [gamma2653.github.io](https://gamma2653.github.io/)

## About This Portfolio

I'm Christopher De Jesus, a Cornell University graduate with a background in Psychology and Computer Science. My work explores how information is processed in both human minds and in machines. I'm fascinated by computational psychology and the parallels between human cognition and artificial intelligence.

## What You'll Find Here

### Home

The landing page introduces my philosophy: understanding how we process information, perceive the present moment, and learn to live more fully. You'll find links to explore my three main areas of work.

### Software Projects

A collection of applications and tools I've built:

- **[Campaign Master](https://github.com/gamma2653/CampaignMaster)** - An AI-enabled companion app for tabletop RPG game masters, built with React, TypeScript, Tailwind CSS, Python, FastAPI, and Pydantic
- **[Habitica-Resync](https://github.com/gamma2653/habitica-resync)** - An Obsidian plugin that syncs your notes and tasks with Habitica for gamified productivity, built with React, TypeScript, and the Obsidian Plugin API
- **[Magiteknician](https://github.com/gamma2653/magiteknician)** - A community-driven rhythm game inspired by osu!, built with Godot Engine and GDScript
- **[StoryBoardWorkshop](https://github.com/gamma2653/StoryBoardWorkshop)** - A creative tool for storytelling and storyboard planning, built with Java
- **[BlockHider](https://github.com/gamma2653/BlockHider)** - A Minecraft plugin that adds hide-and-seek gameplay, built with Java and the Spigot/Bukkit API
- **[TCAT Service Journal](https://github.com/gamma2653/TCAT_Service_Journal)** - A data analysis framework for transit bus activity patterns using Python and ETL pipelines

### Robotics Projects

My work with [Cornell Cup Robotics](https://github.com/cornell-cup) and professional robotics experience:

- **[C1C0 Scheduler](https://github.com/cornell-cup/c1c0-scheduler)** - The communication hub coordinating C1C0's various subsystems, built with Python
- **[R2 Object Detection](https://github.com/cornell-cup/r2-object_detection)** - Real-time object detection for autonomous robot navigation using TensorFlow and SSD MobileNet V1

- **[C1C0 Sounds](https://github.com/gamma2653/c1c0_sounds)** - An experimental Morse code to audio chirp converter built with Python

## Tech Stack

- **Framework:** [React 19](https://react.dev/) with [TypeScript](https://www.typescriptlang.org/)
- **Routing:** [TanStack Router](https://tanstack.com/router) (file-based routing with memory history for GitHub Pages compatibility)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) with dark mode support
- **UI Components:** [Headless UI](https://headlessui.com/) and [Heroicons](https://heroicons.com/)
- **Bundler:** [Rsbuild](https://rsbuild.dev/) (Rspack-based)
- **Testing:** [Rstest](https://rstest.dev/) with React Testing Library
- **Deployment:** [GitHub Pages](https://pages.github.com/) via GitHub Actions

## Development

```bash
npm install          # Install dependencies
npm run dev          # Start dev server with hot reload
npm run build        # Build for production
npm run preview      # Preview production build
npm run lint         # Run ESLint
npm run format       # Format with Prettier
npm run test         # Run tests
```
